Install from jsDelivr

Load CKCSS for presentation and DevinimJS for interaction. Both URLs are version-pinned.

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/kayacuneyd/ckcss@v0.1.0-beta.1/dist/ckcss.min.css">
<script type="module" src="https://cdn.jsdelivr.net/gh/kayacuneyd/devinimjs@v0.2.0/dist/modules/dv-tabs.js"></script>

Core API

APIUse
BaseComponentReactive custom-element base class.
htmlEscaping template tag; required return type for template().
defineExplicit guarded custom-element registration.
createStoreShared state with subscriptions and optional path filtering.
this.outletPreserves server-rendered children inside a component.
import { BaseComponent, html, define } from 'https://cdn.jsdelivr.net/gh/kayacuneyd/devinimjs@v0.2.0/dist/core.min.js';
class AcmeGreeting extends BaseComponent { initialState() { return { name: this.str('name', 'World') }; } template() { return html`<p>Hello ${this.state.name}</p>`; } }
define('acme-greeting', AcmeGreeting);

Optional application runtime

Import app.min.js only when a page needs async data, forms or routes.

import { createAsyncState, fetchJson, createForm, createHashRouter } from 'https://cdn.jsdelivr.net/gh/kayacuneyd/devinimjs@v0.2.0/dist/app.min.js';
const products = createAsyncState(); products.run(() => fetchJson('/api/products.json'));
const checkout = createForm({ email: '' });
const router = createHashRouter().add('/', 'home').add('/products/:id', 'product'); router.start();

Security and accessibility

Interpolated template values are escaped. Use unsafe() only for HTML that you have independently sanitized. Prefer native controls; every interactive component should define keyboard behavior, focus behavior and ARIA semantics.

Run npm run verify before release. It includes lint, unit tests, real-browser tests and the core size gate.
