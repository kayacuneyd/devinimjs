v0.5.1 · build-free runtime

Var olan web’iniz için reactive UI.

DevinimJS, server-rendered HTML’i native browser JavaScript ile erişilebilir ve reactive arayüzlere dönüştürür. Kullanıcınız için Node.js, bundler veya framework runtime gerekmez.

Tek import ile başlaDokümanları oku

PHP, static hosting, WordPress, Laravel ve AI destekli geliştirme için.

Neden DevinimJS?

Deployment basit kalsın. Arayüz canlı olsun.

HTML yapıyı, DevinimJS etkileşimi, CKCSS ise sunumu üstlenir. Framework sözleşmesi küçük, davranışlar açıklanabilir.

01

Build-free sözleşme

Native ES modules’ü sunucunuzdan veya pinned CDN’den yayınlayın.

02

Öngörülebilir reactive state

Proxy state, deterministic template ve odaklı DOM güncellemeleri.

03

Light DOM

CKCSS ve global stiller component markup’ına doğal biçimde ulaşır.

04

Server-friendly

PHP veya HTML data-* ile başlangıç verisini sağlasın; component dv:* event yayınlasın.

05

Erişilebilir primitives

Native kontroller, ARIA, klavye ve focus davranışları sözleşmenin parçasıdır.

06

AI-readable kurallar

Manifest, sabit sözleşmeler ve kopyalanabilir örnekler ajanların doğru üretmesini sağlar.

main veya unversioned CDN URL kullanma.
