Start with HTML. Add behavior when you need it.

DevinimJS is a native browser ES module runtime. Begin with a custom element and a small component contract; add the optional application helpers only when the page needs async data, forms or routing.

Follow the learning path Compare examples 
<script type="module" src="/assets/devinim/authoring.min.js"></script>
<acme-counter data-start="4" data-step="2"></acme-counter>

Serve the module as ordinary JavaScript. No bundler, transpiler or Node.js process is required for consumers.

Core contract

Use component() for ordinary components

The recommended authoring path is a standard .dv.js module. Keep external configuration in typed props, mutable UI data in state, attribute synchronization in sync, user actions in actions, and escaped markup in view.

import { component, html } from '/assets/devinim/authoring.min.js';

component('acme-counter', {
  props: { start: 0, step: 1 },
  state() { return { count: this.props.start }; },
  sync: { start(value) { this.state.count = value; } },
  actions: {
    increment() {
      this.state.count += this.props.step;
      this.emit('change', { count: this.state.count });
    },
  },
  view() {
    return html`<button type="button" on:click="increment">${this.state.count}</button>`;
  },
});

Use on:event="action" for new code. Interpolated values are escaped by the html tag. Components emit bubbling dv:* events with this.emit().

Optional application runtime

Grow a page into an application

Import /app or the corresponding distribution module only when you need application-level coordination. The page or application owns persistence; components expose state and events.

createStore()

Shared reactive state and subscriptions across components.

createAsyncState()

Stale-safe loading, success, empty and error lifecycle for promise-backed work.

fetchJson()

JSON requests for an API or a PHP endpoint.

createForm() / createHashRouter()

Form values, errors and submission state; serverless hash routes for static hosting.

Architecture rule

Know what the runtime does not own

DevinimJS can run client-only applications with browser storage supplied by your page, but it does not provide a database, authentication, payment service or server-side rendering.

FlowWorks client-only?What the project owns

Safety boundary

Keep untrusted content escaped

Use html`` for normal templates. unsafe() is an explicit raw-HTML boundary and is appropriate only for independently sanitized content. API values, user input and future editor output are untrusted.
