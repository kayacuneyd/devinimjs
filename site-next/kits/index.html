Starter kits

A working page, not just one component.

A kit composes a real CKCSS pattern with several DevinimJS components into one working screen — the fastest way to get a real application surface without generating one from scratch.

Generate it

One command, no build step, copies the exact files below into your own project:

node scripts/create-project.mjs my-app --kit=admin-dashboard

Every component below stays on its documented data-*/event contract — nothing here is a hidden framework feature.

Admin dashboard

A CRUD project list, structurally based on CKCSS's data-management.html pattern: search, sort and paginate through <dv-data-table>, create through <dv-modal> + <dv-field>, delete through <dv-confirm>, with <dv-toast-stack> feedback on both. Try it live below — this is the real generated output, not a screenshot.

Projects

Known limitation, by design: <dv-data-table> renders plain-text cells only, no per-row action slot — that's why delete lives in its own small panel instead of a button inside each row. See kits/admin-dashboard/README.md in the repository for the full component list and customization notes.
