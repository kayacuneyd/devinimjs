Referans

Küçük API'ler. Açık sözleşmeler.

Native browser modülleriyle reaktif arayüzler geliştirmek için pratik bir referans.

Tek import ile başla

<link rel="stylesheet" href="/assets/ckcss.min.css"> <script type="module" src="/assets/devinim/core.js"></script>

Geliştirme sırasında source modüllerini, production'da ise sabitlenmiş optimize dağıtımı kullanın.

Authoring yolu

Gündelik bileşenler için component() kullanın. Props, state, action'lar ve view tanımlayın. İleri lifecycle entegrasyonu gerektiğinde BaseComponent kullanın.

Uygulama runtime'ı

Sayfa uygulamaya dönüştüğünde async state, JSON istekleri, formlar, shared store'lar ve hash routing'i devreye alın.

Erişilebilirlik

Native kontrolleri tercih edin. Her etkileşimli bileşen için label, role, focus davranışı, klavye etkileşimi ve live region'ları belgeleyin.

Güvenlik

Interpolated değerler varsayılan olarak escape edilir. API verisini ve editör içeriğini güvenilmeyen kabul edin; raw HTML'i yalnızca bağımsız sanitization sonrasında kullanın.
