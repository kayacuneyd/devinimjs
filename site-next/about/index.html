About the ecosystem

Build for the web people already have.

DevinimJS and CKCSS are designed for teams that want modern interfaces without turning every project into a build-system project.

Our design philosophy

Semantic HTML owns structure. CKCSS owns presentation. DevinimJS owns interaction and state. Each layer stays explainable, portable and useful on its own.

Build-free first

Copy published assets to static hosting, PHP, WordPress or a VPS. Build tooling may optimize publishing; it never defines the consumer contract.

Accessible by default

Native controls, keyboard behavior, visible focus, reduced motion and readable contrast are part of the quality bar.

Who it is for

Independent developers, agencies and small teams building client sites, dashboards, portals, ecommerce interfaces and server-rendered applications.

Project information
